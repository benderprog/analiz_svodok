# Обзор

MVP предназначен для анализа справок в формате DOCX. Пользователь загружает файл, система извлекает ключевые атрибуты (время, подразделение, нарушителей), ищет соответствия в БД портала и показывает результат сравнения.

Основные возможности:
- Экран логина
- Загрузка DOCX и запуск анализа
- Прогресс обработки
- Результаты с деталями сравнения и найденными записями
- Справка, отображаемая из Markdown
- /health — проверка доступности БД/Redis и конфигурации модели
